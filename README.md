# Picture Diary (나만의 그림일기)

Upload a photo and turn it into a childhood memory! 
This simple web app converts your uploaded images into crayon-style drawings and lets you write a diary entry with a handwriting font.

## Features
- **Crayon Filter:** Uploaded images are processed to look like crayon drawings using Canvas API and SVG filters.
- **Handwriting Font:** Uses 'Gaegu' font for a authentic picture diary feel.
- **Notebook Style:** UI designed to mimic a primary school picture diary.

## How to Run
1. Open `index.html` in a modern web browser.
2. Or use a local development server (e.g., Live Server in VS Code, or `python3 -m http.server`).

## Technologies
- HTML5 Canvas & SVG Filters
- CSS3 (Variables, Flexbox)
- JavaScript (Vanilla)
- Google Fonts (Gaegu)
